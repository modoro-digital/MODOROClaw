# Marketing Skills — Thư viện kỹ năng marketing

Bot tự động đọc file SKILL.md trong folder tương ứng khi CEO yêu cầu. Mỗi skill là 1 quy trình chuyên gia — bot follow step-by-step.

## Cách dùng

CEO nhắn bất kỳ yêu cầu nào liên quan marketing → bot đọc INDEX này → chọn skill phù hợp → đọc SKILL.md → follow.

## Danh sách skill

### Viết content & Copy
| Skill | Folder | Khi nào dùng |
|---|---|---|
| Viết copy (landing page, homepage, giá) | `copywriting/` | CEO yêu cầu viết bài bán hàng, headline, CTA, value proposition |
| Chỉnh sửa copy | `copy-editing/` | Sửa, cải thiện copy đã có |
| Nội dung mạng xã hội | `social-content/` | Viết post LinkedIn, Facebook, Instagram, TikTok, Twitter |
| Chiến lược nội dung | `content-strategy/` | Lên kế hoạch content dài hạn, content calendar |
| SEO cho AI (AI-SEO) | `ai-seo/` | Tối ưu nội dung để AI search (ChatGPT, Gemini) trích dẫn |
