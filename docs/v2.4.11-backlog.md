## 4. NSIS same-version installer trap (operational)

NSIS skips install if version unchanged. v2.4.10 was rebuilt multiple times without version bump and required manual mirror-copy of `resources/` to test.
**Rule (now in CLAUDE.md global):** bump version on every new build. Never rebuild the same version in production flow.
