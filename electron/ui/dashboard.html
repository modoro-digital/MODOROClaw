Xóa sạch dữ liệu

Hành động này sẽ xóa TOÀN BỘ:
  • Workspace (AGENTS, knowledge, memory, logs)
  • Config bot (openclaw.json, extensions)
  • Zalo session (cookies, friend cache)
  • Telegram token + chat ID
  • 9Router (API keys, provider config)
  • Google Calendar OAuth token
Không thể khôi phục sau khi xóa. Sau khi reset, app sẽ restart và chạy lại wizard onboarding từ đầu.
Nhập xóa để xác nhận:
Gõ "xóa" (hoặc "xoa" nếu không gõ được dấu)
