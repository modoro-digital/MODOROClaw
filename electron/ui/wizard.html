Lĩnh vực & phong cách

Trợ lý sẽ được tối ưu cho ngành của bạn.

Kết nối trí tuệ nhân tạo

Trợ lý cần được kết nối với AI để hoạt động. Làm theo 3 bước:

1

Tạo tài khoản Ollama (miễn phí):

→ Nhấn nút bên dưới → trang Ollama mở ra

→ Nhấn Sign In (đăng nhập bằng Google/GitHub)

2

Lấy API Key:

→ Sau khi đăng nhập, nhấn nút bên dưới

→ Trang API Keys mở ra → nhấn New API Key

→ Đặt tên gì cũng được → nhấn Create

→ Copy dòng key vừa tạo → dán vào ô bên dưới

3

Nhấn nút bên dưới để hoàn tất thiết lập AI.

Hệ thống sẽ tự động kết nối Ollama và tạo combo fallback.

Sau này muốn thêm AI mạnh hơn? Mở trang quản lý AI bên dưới → thêm Codex, iFlow, hoặc bất kỳ provider nào → thêm model vào combo "main".

Đã mở trang quản lý

📱 Kết nối Telegram

Telegram là nơi bạn nhận thông báo và điều khiển trợ lý.

1

Mở app Telegram trên điện thoại hoặc máy tính

→ Tìm kiếm @BotFather

→ Gửi tin nhắn: /newbot

→ Đặt tên cho bot (ví dụ: "Trợ Lý MODORO")

→ BotFather sẽ gửi cho bạn một dòng chữ và số rất dài (dạng 123456:ABC...) — copy toàn bộ dòng đó

2

Vẫn trong Telegram:

→ Tìm kiếm @userinfobot

→ Gửi tin nhắn: /start

→ Bot trả về một dãy số (chỉ gồm số, ví dụ: 1196242919) — copy dãy số đó

☝️

Quan trọng: Trước khi test, hãy:

→ Mở Telegram → tìm bot bạn vừa tạo

→ Gửi tin nhắn /start cho bot đó

→ Rồi quay lại đây nhấn "Test"
